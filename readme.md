# API Projeto

Este é o repositório para a API do projeto.

## Pré-requisitos

- Node.js (versão 12 ou superior)
- NPM (gerenciador de pacotes do Node.js)

## Instalação

Para instalar as dependências do projeto, execute o comando abaixo:

```bash
npm install
```

Para inciar o servidor:

```bash
npm run start
